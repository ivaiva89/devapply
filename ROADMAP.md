# Product Roadmap

## Phase 1 --- MVP

Goal: Launch the Job Application Tracker quickly.

Features

-   authentication
-   application CRUD
-   applications table
-   kanban pipeline
-   dashboard metrics
-   reminders
-   resume attachments
-   billing
-   analytics basics

Success metrics

-   active users
-   first paying customers
-   feature usage validation

## Phase 2 --- Developer Job CRM

Goal: Improve retention and workflow depth.

Features

-   recruiter contacts
-   company profiles
-   communication history
-   interview timeline
-   follow‑up tasks

## Phase 3 --- Career Toolkit

Goal: Expand into a developer career platform.

Products

-   Resume Builder
-   Cover Letter Generator
-   Interview Preparation Tracker
-   Salary Comparison Tool
