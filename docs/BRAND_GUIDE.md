# DevApply Brand Guide

## Brand Position

DevApply is a developer-focused SaaS for managing job applications and
career workflows.

The product should feel:

- modern
- structured
- developer-centric
- calm and professional
- productivity-focused

Inspired by the visual clarity of tools like **Linear, Vercel, GitHub,
and Raycast**.

---

# Core Brand Message

**DevApply helps developers run their job search like a professional
workflow.**

Primary tagline:

> The job tracker built for developers.

Alternative taglines:

- Run your job search like a workflow
- Track every application. Improve every step
- Your developer job search, organized

---

# Logo Concept

Primary symbol:

    {✓}

Meaning:

- `{ }` represents **code**
- `✓` represents **application success / progress**

Logo requirements:

- minimal
- geometric
- vector-based
- recognizable as favicon
- modern SaaS aesthetic

Deliverables:

- logo mark
- wordmark
- icon only version
- light and dark versions
- SVG master

# Landing Page Style

Principles:

- minimal UI
- generous whitespace
- restrained colors
- subtle borders instead of heavy shadows
- developer-focused layout

## Page Structure

1.  Hero
2.  Feature cards
3.  Product preview screenshots
4.  Workflow explanation
5.  Final call-to-action

Example hero copy:

**Track your developer job search in one place**

Manage applications, interviews, follow-ups, and pipeline progress with
a workflow built for developers.

CTA:

- Start tracking for free
- See how it works

---

# UI Design Rules

## Buttons

- rounded-xl
- primary color = indigo
- secondary = outline

## Cards

- rounded-2xl
- subtle borders
- minimal shadows

## Inputs

- clear focus ring
- neutral background

## Icons

Use **Lucide icons** for consistency.
